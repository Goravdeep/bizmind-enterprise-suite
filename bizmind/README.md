## 🎯 Overview

**BizMind Enterprise Suite** is an AI-powered business management platform that leverages **Microsoft Foundry SDK** for multi-step reasoning, **Microsoft Graph API** for intelligent email alerts, **Microsoft Entra ID** for enterprise-grade authentication, and **Microsoft Work IQ** for document intelligence.

### ✨ Key Features

| Feature | Description | Microsoft Tool |
|---------|-------------|----------------|
| 🧠 AI Reasoning Agent | Multi-language (English/Hindi/Hinglish) with step-by-step reasoning | Microsoft Foundry SDK |
| 📧 Smart Alerts | Automated email notifications for low stock & profit thresholds | Microsoft Graph API |
| 🛡️ Enterprise Auth | Secure authentication ready for Azure AD | Microsoft Entra ID |
| 📄 Document IQ | Intelligent document analysis and data extraction | Microsoft Work IQ |
| 🏭 Warehouse Management | Complete inventory & supplier management | Custom + SQLite |
| 💰 Financial Tracking | Income/Expense tracking with PDF reports | Custom + PDFKit |
